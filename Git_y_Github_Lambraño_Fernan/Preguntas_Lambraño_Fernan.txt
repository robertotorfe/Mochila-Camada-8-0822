¿Quién inventó el sistema de control de versión Git y por qué?

Es un software de control de versiones diseñado por Linus Torvalds creador del kernel de Linux el cual tenía un gestor de paquetes que pasó a ser de pago, por lo que se decidio crear un nuevo gestor de control de versiones para su sistema, y que diera soporte a su core. De esta forma nació Git.
