b.¿A quién pertenece actualmente Github y por qué?
**Respuesta: El 4 de junio de 2018, Microsoft compró GitHub. La compra de GitHub es un paso más en la inclinación de Microsoft para enfocarse todo lo posible al desarrollo, haciendo una vez más hincapié en que el núcleo de la empresa lo forman los desarrolladores y nada más.
