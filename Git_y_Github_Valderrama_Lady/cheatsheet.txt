Subir cambios a la rama 
 
Git branch 	Verificar SIEMPRE la rama donde estoy ubicada 
Git remote 	¿? 
Git pull 	Se utiliza para bajar lo cambios SIEMPRE 
Git status 	Verificar que los cambios que se van subir son los que yo modifique 
Git add . 	Subir lo cambios 
Git commit -m “Aquí va el comentario” 	Confirmar cambios con su respectivo comentario
Ejemplo: "Finish Consulta_consolidado_inversiones,Consulta_consolidado_prestamos, Consulta_consolidado_productos_dda" 
Al final debe decir la cantidad de archivos modificados: 
Ejemplo: 3 files changed, 9 insertions(+), 9 deletions(-) 
Git push origin rama 	Subir cambios al repositorio en mi rama 
Ejemplo: git push origin feature/estabilizacionLady
